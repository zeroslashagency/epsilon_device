# web_dashboard

React + Vite dashboard for monitoring device status from Supabase.

## Setup

Create `.env.local` in this folder:

```bash
VITE_SUPABASE_URL=https://<project-ref>.supabase.co
VITE_SUPABASE_ANON_KEY=<your-anon-or-publishable-key>
```

## Run

```bash
npm install
npm run dev
```

## Build and lint

```bash
npm run lint
npm run build
```
